## Installation

1. **Clone the repository:**
    
    ```bash
    git clone https://github.com/Shaban-Eissa/Jisr-React-Task
    ```
    
2. **Install dependencies:**
    
    ```bash
    cd Jisr-React-Task
    npm install
    ```

3. **Run the application:**
    
    ```bash
    npm start
    ```
    
    The application will be accessible at `http://localhost:3000`.
